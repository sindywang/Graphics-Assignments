How to setup Visual Studio Express to work with OpenGL on Windows

19/01/201426/01/2014marcelocra C, C++, Configuration, Installation, Tutorial, Windows Tags: c, c++, configure, express, freeglut, gl, install, open, opengl, setup, tutorial, visual studio

Requirements

Visual Studio Express

The first thing to do, if you already haven’t done, is install the Visual Studio Express software. Probably this tutorial also works for the other versions of Visual Studio, but I only tested it with Express. You can find the links to the installation here. The installation process is quite straightforward, so once you are done you can launch the program and proceed to the next part.

FreeGlut

Download and extract the Freeglut library from here. Since we are using Visual Studio, go to the section freeglut 2.8.1 MSVC Package of the website. Remember the location to where you extracted the Freeglut folder, because it will be necessary in the next steps.

Initializing an OpenGL program

Open Visual Studio Express and select New Project… from the left part of the startup window. Another window will open (check figure 1) and now you need to select Visual C++ (inside Templates which is inside Installed), and then Empty Project. Name it, choose where to save and most importantly be sure to check Create directory for solution (otherwise you folder will become a mess).

Figure 1

Afer the project is created, right click on the name you gave to the project (figure 2) and go to Properties.

Figure 2

A new window will appear with a lot of options. I’ll list what you need to do in this window:

  • In the Configuration Properties, go to VC++ Directories. You will see a lot of directories names. Click on Include Directories, then on the dropdow arrow that appears to the right and then on Edit…. In the new window, select the new folder icon (which says ‘New line’ if you hover with the mouse) and in the new line that appears, click on the … (three dots by the end of the line). You will get a new window to select the folder where your include files are located. Now go to the freeglut folder you extracted in the beginning of the tutorial and select the include folder inside that folder.
  • Still on VC++ Directories, click on Library Directories and do the same as before, but now select the lib folder inside of the freeglut folder.
  • Now open the Configuration Properties sub-item Linker and then click on Input. Click on Additional Dependencies and again on the dropdown arrow and Edit…. It will appear a new window, but in this one you will just type opengl32.lib and click Ok. After this you can click Ok and that should close the Properties window.

You now have your visual studio configured and you need to test it. Right click on Source Files inside your solution folders and go to Add and then New Item…. Choose a C++ File, give it a name and then click Add. Copy paste the code below there and run. This code is a robot arm that you can move with ‘s’ and ‘e’ keyboard keys. You can find this code and a lot more on the Red Book, a great OpenGL programming guide for those who don’t know OpenGL.


#include <GL/glut.h>

static int shoulder = 0, elbow = 0;

void init(void)
{
	glClearColor(0.0, 0.0, 0.0, 0.0);
	glShadeModel(GL_FLAT);
}

void display(void)
{
	glClear(GL_COLOR_BUFFER_BIT);
	glPushMatrix();
	glTranslatef(-1.0, 0.0, 0.0);
	glRotatef((GLfloat)shoulder, 0.0, 0.0, 1.0);
	glTranslatef(1.0, 0.0, 0.0);
	glPushMatrix();
	glScalef(2.0, 0.4, 1.0);
	glutWireCube(1.0);
	glPopMatrix();

	glTranslatef(1.0, 0.0, 0.0);
	glRotatef((GLfloat)elbow, 0.0, 0.0, 1.0);
	glTranslatef(1.0, 0.0, 0.0);
	glPushMatrix();
	glScalef(2.0, 0.4, 1.0);
	glutWireCube(1.0);
	glPopMatrix();

	glPopMatrix();
	glutSwapBuffers();
}

void reshape(int w, int h)
{
	glViewport(0, 0, (GLsizei)w, (GLsizei)h);
	glMatrixMode(GL_PROJECTION);
	glLoadIdentity();
	gluPerspective(65.0, (GLfloat)w / (GLfloat)h, 1.0, 20.0);
	glMatrixMode(GL_MODELVIEW);
	glLoadIdentity();
	glTranslatef(0.0, 0.0, -5.0);
}

void keyboard(unsigned char key, int x, int y)
{
	switch (key) {
		case 's':   /*  s key rotates at shoulder  */
			shoulder = (shoulder + 5) % 360;
			glutPostRedisplay();
			break;
		case 'S':
			shoulder = (shoulder - 5) % 360;
			glutPostRedisplay();
			break;
		case 'e':  /*  e key rotates at elbow  */
			elbow = (elbow + 5) % 360;
			glutPostRedisplay();
			break;
		case 'E':
			elbow = (elbow - 5) % 360;
			glutPostRedisplay();
			break;
		default:
			break;
	}
}

int main(int argc, char** argv)
{
	glutInit(&argc, argv);
	glutInitDisplayMode(GLUT_DOUBLE | GLUT_RGB);
	glutInitWindowSize(500, 500);
	glutInitWindowPosition(100, 100);
	glutCreateWindow(argv[0]);
	init();
	glutDisplayFunc(display);
	glutReshapeFunc(reshape);
	glutKeyboardFunc(keyboard);
	glutMainLoop();
	return 0;
}

With this we end the tutorial. It took me a long time until I could make OpenGL run, specially because I tried before with other IDEs, like Dev, and it simply didn’t work. I hope you can run it with this instructions. See you later!
